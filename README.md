# Art Viewing Assistant

本システムは、絵画鑑賞において鑑賞者が抱いた印象や気づきを、
対話を通じて言語化・整理することを支援するWebアプリケーションです。

## 概要
本アプリケーションは、鑑賞初心者が「何をどう見ればよいかわからない」
と感じやすい絵画鑑賞の場面において、  
段階的な対話インターフェースを通じて、

- 初期印象の言語化
- 印象の根拠の整理
- 別視点の提示と再解釈
- 最終的な解釈の統合

を支援することを目的としています。

## 主な機能
- 画像アップロードによる絵画入力
- チャット形式による鑑賞対話
- 印象の根拠整理（Step1）
- 画像認識に基づく別視点提示（Step2）
- 解釈の統合と文章化（Step3）

## 使用技術
- Next.js
- React
- TypeScript
- OpenAI API（画像入力対応モデル）
